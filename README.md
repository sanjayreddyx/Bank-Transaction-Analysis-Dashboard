# 💳 Bank Transaction Analysis Dashboard

An interactive **Power BI Dashboard** built to analyze UPI bank transaction data. This dashboard provides business insights through KPIs, interactive charts, filters, and summaries to help understand transaction patterns, banking performance, and merchant activity.

---

## 📌 Project Overview

This dashboard analyzes UPI transaction records and presents meaningful insights such as:

- Total Transaction Value
- Total Credit & Debit Amount
- Average Transaction Value
- Total Transactions
- Bank-wise Performance
- Merchant-wise Analysis
- Credit vs Debit Distribution
- Business Insights Summary
